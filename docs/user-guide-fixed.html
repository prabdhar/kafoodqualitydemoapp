üè´ School Management

Viewing Schools

  1. Click "Manage Govt Schools" from dashboard or navigation
  2. Browse through the list of registered schools
  3. Use search and filter options to find specific schools

School Information Displayed

  • School Name and location
  • School Type (Government School, Government Aided)
  • Category (Primary, Higher Secondary, High School)
  • Status (Active, Under Review, Suspended)
  • Last Inspection Date
  • Overall Rating (A, B+, B, C, D)
  • Contact Information

üîç Inspection System

Starting an Inspection

Method 1: From Dashboard

Click "New Govt School Inspection" quick action ‚Üí Full-page inspection form opens

Method 2: From School Management

Go to "Manage Govt Schools" ‚Üí Click "Inspect" button on specific school

Method 3: From Navigation

Click "New Inspection" in navigation menu ‚Üí Full-page inspection form opens

Inspection Form Features

  • School Selection: Dropdown with all registered schools
  • Inspector Name: Required field
  • Inspection Date: Defaults to current date
  • Inspection Type: Routine, Complaint-based, Follow-up, Surprise
  • Overall Rating: A (Excellent) to D (Poor)
  • Findings: Detailed inspection observations
  • Violations: Any non-compliance issues found
  • Recommendations: Improvement suggestions

‚öôÔ∏è Admin Panel

Admin Access Required: Only users with Administrator role can access these features.

Portal Logo Management

  1. Login with admin credentials (admin/admin123)
  2. Click "Admin Panel" in navigation menu
  3. Find "Upload Facility Photos" section
  4. Select facility type if needed
  5. Upload logo file (PNG, JPG, SVG up to 2MB)
  6. Preview logo before saving
  7. Click "Save Logo" to apply changes

Logo Guidelines

  • Recommended Size: 200x80 pixels or similar aspect ratio
  • Format: PNG with transparent background for best results
  • File Size: Maximum 2MB
  • Usage: Logo appears in navigation bar and login page

üö® Troubleshooting

Login Problems

Issue: Cannot login with provided credentials

Solutions:

  • Verify username and password are correct
  • Check caps lock is off
  • Try refreshing the page
  • Clear browser cache and cookies

Photo Upload Issues

Issue: Photos won't upload

Solutions:

  • Ensure photos are under 2MB
  • Use PNG, JPG, or SVG formats only
  • Select a school before uploading
  • Check internet connection
