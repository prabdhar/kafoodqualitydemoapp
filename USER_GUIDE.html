1. Application Overview

The Karnataka Government School Food Safety Application is a comprehensive digital platform designed to streamline food safety inspections across all government schools in Karnataka state. This application enables food safety inspectors to conduct thorough inspections, document findings with photos, manage school data, and track safety programs.

Key Features

  • Digital Inspections: Comprehensive multi-step inspection forms
  • Photo Documentation: Upload and manage inspection photos
  • School Management: Manage 8,547+ Karnataka government schools
  • Safety Programs: Track training and certification programs
  • Real-time Dashboard: Monitor compliance and inspection status
  • Reports & Analytics: Generate detailed inspection reports
Target Users: Food Safety Inspectors, Education Department Officials, School Administrators, and Program Coordinators working with Karnataka government schools.

2. Getting Started

System Requirements

  • Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • Internet: Stable internet connection required
  • Device: Desktop, laptop, or tablet (mobile responsive)
  • Screen Resolution: Minimum 1024x768 recommended

Accessing the Application

Step 1: Open your web browser and navigate to the application URL
http://localhost:3000 (for development) or your deployed URL
Step 2: You will be automatically redirected to the login page if not authenticated
Note: Ensure you have received your login credentials from your system administrator before attempting to access the application.

5. School Management

Accessing School Management

Navigate to Schools from the main menu to access the Karnataka Government School Management interface.

School Information

Each school record contains:

  • School Name: Official name of the government school
  • Type: Government School classification
  • Owner: Karnataka Education Department
  • Location: District and area within Karnataka
  • Contact: Phone and email information
  • License Number: KGS-YYYY-XXX format
  • Safety Rating: Current food safety rating
  • Inspection Dates: Last and next scheduled inspections
  • Violations: Number of current violations

Search and Filter

Search: Use the search bar to find schools by name, location, or owner
Filter by Status: Filter schools by Active, Under Review, or Suspended status

School Categories

  • Primary School: Classes 1-5
  • Higher Secondary: Classes 1-10
  • Higher Secondary: Classes 1-12
Tip: Use the search functionality to quickly locate specific schools before conducting inspections.

6. Food Safety Inspections

Starting a New Inspection

Step 1: Click "New Govt School Inspection" from the dashboard or navigation menu
Step 2: You'll be taken to the multi-step inspection form

Inspection Form Sections

Section 1: Government School Details

  • Inspection Type (Routine, Complaint-based, Follow-up, Licensing)
  • School Name
  • School Type (Government Primary, High School, Higher Secondary, Residential)
  • Location and District
  • State (Karnataka - pre-filled)
  • Inspection Date
  • Inspector Name

Section 2: License & Contact Information

  • License Number (KGS format)
  • Principal Name
  • Contact Number

Section 3: Food Safety Assessment

  • Hygiene Rating
  • Temperature Check
  • Storage Conditions
  • Kitchen Staff Hygiene
  • Equipment Cleanliness
  • Waste Management

Section 4: Inspection Findings

  • Detailed Findings
  • Recommendations

Section 5: Photo Documentation

  • Upload inspection photos
  • Photo management and preview

Section 6: Final Report

  • Document uploads
  • Final submission

Navigation Between Sections

Next: Click "Next" to proceed to the following section
Previous: Click "Previous" to return to the previous section
Save Draft: Save your progress without submitting
Submit: Complete and submit the inspection report
Important: Ensure all required fields are completed before attempting to submit the inspection form.

7. Photo Documentation

Photo Upload Features

The application includes a comprehensive photo upload system for documenting inspection findings.

Supported Features

  • Drag & Drop: Drag photos directly into the upload area
  • File Browser: Click to browse and select files
  • Preview: View uploaded photos before submission
  • File Validation: Automatic file size and type checking
  • Delete: Remove unwanted photos

Upload Process

Step 1: Navigate to the "Photo Documentation" section of the inspection form
Step 2: Either drag photos into the upload area or click "Browse Files"
Step 3: Review uploaded photos in the preview area
Step 4: Add captions or descriptions if needed
Step 5: Remove any unwanted photos using the delete button

File Requirements

  • Supported Formats: JPG, JPEG, PNG, GIF
  • Maximum File Size: 5MB per photo
  • Maximum Photos: 20 photos per inspection
  • Recommended Resolution: 1920x1080 or higher
Best Practice: Take clear, well-lit photos that clearly show the areas being inspected. Include photos of kitchen areas, storage facilities, and any violations found.

10. Troubleshooting

Common Issues and Solutions

Login Issues

Problem: Cannot log in with provided credentials
Solution: Verify username and password are correct. Check caps lock. Contact administrator if issue persists.

Photo Upload Issues

Problem: Photos won't upload
Solution: Check file size (max 5MB), file format (JPG, PNG), and internet connection. Try refreshing the page.

Form Submission Issues

Problem: Cannot submit inspection form
Solution: Ensure all required fields are completed. Check internet connection. Save as draft and try again.

Performance Issues

Problem: Application running slowly
Solution: Clear browser cache, close unnecessary tabs, check internet speed, try different browser.

Browser Compatibility

  • Recommended: Chrome 90+, Firefox 88+
  • Supported: Safari 14+, Edge 90+
  • Not Supported: Internet Explorer

System Requirements Check

Step 1: Ensure stable internet connection (minimum 1 Mbps)
Step 2: Update browser to latest version
Step 3: Enable JavaScript and cookies
Step 4: Disable ad blockers if experiencing issues
